# Wildernis

Wat is dat?  Klinkt leuk!  Wil ik heen!

Kijk om je heen - je ziet wel ergens een stukje wildernis.  Waarschijnlijk
begint het met groen.  Loop er eens heen en voel eraan.  Zacht?  Nat?  Prikt het?  Waar groeit het op?  Heeft het vrienden?  Beweegt het?

Kijk eens dichter.  Can you lick it?
