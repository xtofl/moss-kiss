In Vlaanderen is maar weinig wildernis, dus is ze heel waardevol.  Je zorgt voor wat je graag hebt.  Daarom wil ik met mos-kus mensenharten openen om de wildernis binnen te laten.

Zelf ben ik een all-rounder: van alles wel wat.  Dat betekent dat ik voor
iedereen wel iets interessants kan bieden.  Verdieping - daarvoor kan ik je
dan doorverwijzen naar opleidingen bij collega's.
