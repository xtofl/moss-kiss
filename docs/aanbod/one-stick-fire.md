# One-stick fire

Maar het regent.  Het bos is nat.  Alles is nat.  Hoe kan ik nu
een vuur maken?

We leren hoe we een geschikt stuk hout vinden, dat we verwerken
zodat we het juiste materiaal hebben om een vuur aan te steken.  We
gebruiken zaag, mes en fire-steel.

Na deze workshop weet je in barre omstandigheden vuur aan te krijgen.
