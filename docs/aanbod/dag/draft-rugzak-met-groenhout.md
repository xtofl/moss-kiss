# Een rugzak van groenhout

Hoe draag je dingen rond in het bos?

Moet je echt zo'n dure rugzak kopen?

Op deze dagworkshop oogsten we groenhout en bast.  Met deze materialen
gaan we aan de slag om een stevig frame te maken dat we op onze rug
kunnen dragen.

Dit frame leren we bepakken met de spullen die we nodig hebben voor
een kleine trektocht.

Na deze workshop kan je met je zelfgemaakte rugzak op pad.
